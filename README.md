# chatbot-kesehatan-mental
Repository ini berisi kode sumber pengembangan chatbot kesehatan mental yang dibuat sebagai bagian dari skripsi berjudul:
“Pengembangan Chatbot untuk Kesehatan Mental di Era Generasi Z Menggunakan Framework Rasa”

Chatbot dikembangkan untuk memberikan dukungan awal dan informasi terkait kesehatan mental kepada pengguna, khususnya Generasi Z, melalui interaksi berbasis teks dengan pendekatan Natural Language Processing (NLP).

Teknologi yang Digunakan :
- Python
- Rasa Open Source
- Natural Language Processing (NLP)
- Telegram Bot API

Cara Menjalankan Sistem :
- Pastikan Python telah terpasang.
- Instal dependensi: pip install rasa
- Jalankan action server: rasa run actions
- Jalankan server Rasa: rasa run
Integrasi dengan Telegram memerlukan token bot yang tidak disertakan dalam repositori ini.
